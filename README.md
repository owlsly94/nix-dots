# Nixos flake & dots
To install the system:
```
sudo nixos-rebuild switch --flake .#OwlslyBox
```

## Usage of template flakes
### Python example
Create projects dir and enter it:
```
mkdir projects
cd projects
```
Create a test dir and enter it:
```
mdkir test-python
cd test-python
```
Here just type the zsh alias:
```
devpy
```
There's also alias for web-dev `devweb`
